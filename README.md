# GitPilot

**AI-powered Git GUI for macOS**

GitPilot is a desktop Git client that acts as your always-on command center — monitoring all your repositories, generating commit messages with AI, and letting you review PRs across GitHub and GitLab from one place.

## Download

**[Download Latest Release](https://github.com/RbrtDdds/gitpilot-releases/releases/latest)**

- **Apple Silicon** (M1/M2/M3/M4): Download the `arm64` DMG
- **Intel Mac**: Download the `x64` DMG

## Installation

1. Download the DMG for your Mac
2. Open the DMG and drag GitPilot to Applications
3. First launch: **Right-click → Open** (macOS will warn about unsigned app)
4. Go to System Settings → Privacy & Security → Click "Open Anyway" if prompted

## Features

- **AI Commit Messages** — Generate conventional commit messages from your diff (Claude / OpenAI)
- **Branch Explorer** — Click any branch to see commits and AI-generated summary
- **PR Dashboard** — View all PRs across GitHub and GitLab in one place
- **Inline Code Review** — Read and write review comments directly in the diff viewer
- **Notifications** — Background polling with macOS system notifications
- **Menu Bar App** — Runs in the background with a tray icon, quick-view popup
- **Multi-Repo Hub** — Monitor all your repositories from a single app

## Auto-Update

GitPilot checks for updates automatically. When a new version is available, you'll be notified in the app.

## Requirements

- macOS 12 (Monterey) or later
- Git installed (`xcode-select --install`)
